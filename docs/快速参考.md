## 🎯 常见任务

### 添加新问答

1. 打开 `dist/data/knowledge-base.json`
2. 在 `questions` 数组中添加：
```json
{
  "id": 新ID,
  "keywords": ["关键词1", "关键词2"],
  "answer": "回答内容"
}
```
3. 保存并刷新页面

### 修改语音速度

在 `dist/js/speech.js` 中找到：
```javascript
utterance.rate = 1.0;  // 改为 0.5-2.0
```

### 修改着色器速度

在 `dist/js/shader.js` 中找到：
```javascript
iTime * 0.01  // 旋转速度
iTime * 0.15  // 波形速度
```

### 修改音频反应灵敏度

在 `dist/js/audio.js` 中找到：
```javascript
const CONFIG = {
  intensity: 1.5,  // 改为 0.5-3.0
  // ...
};
```

## 📊 性能优化

### 着色器
- 帧率：30 FPS（可在 shader.js 修改）
- 分辨率：自适应窗口大小

### 音频
- FFT 大小：256（可在 audio.js 修改）
- 平滑因子：0.3-0.5

### 知识库
- 建议条目：< 1000
- 关键词数：3-5 个/条目

## 🐛 常见问题

### 没有声音
1. 检查系统音量
2. 检查浏览器权限
3. 查看控制台错误
4. 尝试 `voiceHelper.list()`

### 语音识别失败
1. 检查麦克风权限
2. 确认语言设置为中文
3. 检查网络连接
4. 尝试其他浏览器

### 背景不显示
1. 检查 WebGL 支持
2. 更新显卡驱动
3. 查看控制台错误
4. 尝试其他浏览器

### 知识库不生效
1. 检查 JSON 格式
2. 查看网络请求
3. 确认文件路径正确
4. 刷新浏览器缓存
