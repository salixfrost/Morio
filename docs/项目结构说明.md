## 文件说明

### 核心文件

#### HTML
- `index.html` - 主页面，包含 UI 结构和 SVG 滤镜

#### CSS
- `style.css` - 样式文件，包含动画和布局

#### JavaScript 模块

1. **main.js** - 主入口
   - 初始化所有模块
   - 管理 DOM 元素
   - 协调各模块交互

2. **shader.js** - WebGL 着色器
   - 动态背景渲染
   - 噪声函数
   - 30 FPS 限制

3. **audio.js** - 音频处理
   - 音频播放控制
   - Web Audio API 分析
   - 音频反应性动画

4. **speech.js** - 语音识别
   - 语音输入识别
   - 语音输出合成
   - 知识库查询

5. **blob.js** - Blob 交互
   - 悬停动画加速
   - 动画控制

6. **knowledge.js** - 知识库管理
   - 加载知识库
   - 关键词匹配
   - 模糊搜索

7. **voice-helper.js** - 语音助手
   - 语音包列表
   - 语音测试
   - 推荐语音选择

8. **bundle.js** - 打包版本
   - 所有模块合并
   - 无需模块加载器
   - 兼容性最好

### 数据文件

#### knowledge-base.json
本地知识库，包含问答对：
```json
{
  "questions": [

      "id": 1,
      "keywords": ["关键词"],

    }

}
```

## 版本说明

### dist/ (生产版本)
- 可直接部署到服务器
- 包含打包版本 (bundle.js)
- 包含模块化版本 (js/)
- 推荐使用 bundle.js

### src/ (开发版本)
- 源代码
- 与 dist/ 保持同步
- 用于开发和调试

## 使用方式

### 方式1：使用打包版本（推荐）
```html
<script src="./bundle.js"></script>
```
- 无需模块加载器
- 兼容性最好
- 可直接打开 HTML

### 方式2：使用模块化版本
```html
<script type="module" src="./js/main.js"></script>
```
- 代码结构清晰
- 易于维护
- 需要 HTTP 服务器

## 开发指南

### 修改代码
1. 在 `src/` 目录修改源代码
2. 测试功能
3. 同步到 `dist/` 目录

### 添加新功能
1. 在 `src/js/` 创建新模块
2. 在 `main.js` 中导入
3. 更新 `bundle.js`

### 修改知识库
1. 编辑 `dist/data/knowledge-base.json`
2. 刷新页面即可生效

### 调试
1. 打开浏览器控制台
2. 使用 `voiceHelper` 工具
3. 查看日志输出

## 部署

### 静态部署
直接将 `dist/` 目录部署到：
- GitHub Pages
- Netlify
- Vercel
- 任何静态服务器

### 本地运行
```bash
# Python
python -m http.server 8000

# Node.js
npx serve dist

# PHP
php -S localhost:8000 -t dist
```
